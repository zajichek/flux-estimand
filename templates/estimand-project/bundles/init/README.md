# init

Minimal Flux `ModelBundle` implementation for the `goalie_pull_policy` estimand.

This bundle should estimate the target quantities defined in the root `ESTIMAND.md` using the simplest state representation and processes that are credible under explicit assumptions.

It should document:

- supported conditioning state
- counterfactual goalie-pull policies
- required inputs
- simulation outputs
- assumptions
- validation checks
- known limitations
